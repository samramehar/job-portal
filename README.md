# Job Portal Website (MERN Stack)

A full-stack **Job Portal Web Application** built using the **MERN stack (MongoDB, Express, React, Node.js)**. This project allows users to register, browse jobs, apply for jobs, and manage job listings with an admin dashboard.

---

## Project Overview

This Job Portal system is designed to simulate a real-world recruitment platform where:

- Job seekers can browse and apply for jobs
- Employers/Admin can post and manage job listings
- Authentication is used for secure access
- Role-based access (User/Admin) is implemented

---

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS / Tailwind (if used)

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt

---

## Key Features

### 👤 User Features
- User registration & login
- Browse available jobs
- Apply for jobs
- Save jobs (if implemented)
- Profile management

### Admin Features
- Add new job listings
- Edit / delete jobs
- View applicants
- Manage job posts

### Authentication
- JWT-based authentication
- Password hashing using bcrypt
- Protected routes
